### 📋 已知資訊

| 項目 | 內容 |
|------|------|

### ❓ 待釐清問題（AI 提問）

為了能夠提供完整的設定步驟和 VLAN 規劃，我需要了解以下資訊：

#### 1. 網路架構相關

- [ ] **核心交換器型號**：學校使用什麼品牌/型號的核心交換器？（Cisco、HPE Aruba、其他？）
- [ ] **現有網路拓撲**：是否有網路架構圖？（核心層 → 匯聚層 → 存取層）
- [ ] **AP 連接方式**：AP 是直接接到核心交換器，還是接到各樓層的存取交換器？
- [ ] **上行頻寬**：核心交換器到 Controller 的連線頻寬？

#### 2. Aruba Controller 相關

- [ ] **Controller 型號**：是哪個型號？（如 7010、7030、7205、7220 等）
- [ ] **Controller 版本**：ArubaOS 版本？
- [ ] **Controller IP**：管理 IP 位址？
- [ ] **Controller 部署位置**：放在哪個網段？

#### 3. VLAN 規劃相關

- [ ] **VLAN 24 用途**：目前 VLAN 24 是做什麼用的？（如：教職員無線、學生無線？）
- [ ] **VLAN 25 用途**：目前 VLAN 25 是做什麼用的？
- [ ] **IP 範圍**：每個獨立 VLAN 需要多少 IP？（/24、/25、/26？）
- [ ] **DHCP 配置**：DHCP 伺服器是誰提供？（交換器、Windows Server、其他？）

#### 4. AP 部署相關

- [ ] **AP 分布**：50 台 AP 分布在幾棟建築物/幾個樓層？
- [ ] **AP505 vs AP305 數量**：各有幾台？
- [ ] **命名規則**：是否有 AP 命名規則？（如：B1-3F-AP01 表示 B1 大樓 3 樓第 1 台）
- [ ] **現有 AP 管理**：目前這些 AP 如何管理？已經在 Controller 上嗎？

#### 5. 管理需求

- [ ] **SSID 數量**：要廣播幾個無線網路名稱？
- [ ] **認證方式**：使用什麼認證？（PSK、802.1X、訪客入口？）
- [ ] **頻寬控制**：是否需要針對不同 SSID/VLAN 做頻寬限制？
