# ADR-001: VLAN 策略選擇

> 狀態：🟡 待定（Pending）
> 日期：2026-04-15

---

## 背景

學校有約 50 台 Aruba AP（AP505/AP305）需要部署，要求每台 AP 有獨立 VLAN。需要決定 VLAN 編號策略和 IP 規劃方式。

## 決策

（待用戶提供網路架構資訊後決定）

## 選項

### 選項 A：連續編號（101-150）

**優點：**
- 簡單易管理
- 方便自動化腳本處理

**缺點：**
- 無法從 VLAN 編號識別位置

### 選項 B：依建築物分段

**優點：**
- 從 VLAN 編號可知 AP 所在建築
- 便於分區管理

**缺點：**
- 需要預留空間給未來擴展

### 選項 C：依樓層分段

**優點：**
- 從 VLAN 編號可知樓層
- 適合垂直佈線架構

**缺點：**
- 不同建築可能造成混淆
