# 資訊競賽文件生成工具

每年只要修改一個 YAML 檔案，就能一鍵生成所有需要列印的文件。

---

## 每年的操作流程（3 步驟）

### 第 1 步：更新獲獎名單

編輯 `winners.yaml`，填入當年的資料：

```yaml
year: 116          # ← 改成當年民國年
competition: 116年度中小學資訊網路應用競賽
doc_ref: "中市教課字第116XXXXXXX號"   # ← 填入公文字號

students:
  - name: 王小明
    class: "801"
    seat: "5"
    event: 電腦打字國中英文甲組
    rank: 甲等
    award: 嘉獎壹次
    prize: 150
    teacher: 李老師
  # ... 繼續新增其他學生
```

### 第 2 步：執行生成腳本

```bash
python gen.py
```

或指定特定年度的 YAML 檔案：

```bash
python gen.py winners_116.yaml
```

### 第 3 步：列印

1. 在 `output_115/`（或對應年度）資料夾中找到生成的 `.html` 檔案
2. 用瀏覽器開啟（建議 Chrome 或 Edge）
3. 按 `Ctrl + P` 列印
4. 列印設定：紙張 **A4**、縮放 **100%**、勾選「**背景圖形**」

---

## 生成的文件清單

| 文件 | 說明 |
|------|------|
| `115年_獎懲申請表_XXX.html` | 每位學生一份，需分別列印 |
| `115年_敘獎擬議清冊.html` | 教師敘獎清冊（橫向 A4）|
| `115年_家長會獎學金印領清冊.html` | 家長會獎學金 |

---

## 資料夾結構

```
tool/
├── winners.yaml          # ← 每年修改這個
├── gen.py                # 生成腳本（不需修改）
├── templates/            # HTML 版型（不需修改）
│   ├── 獎懲申請表.html
│   ├── 敘獎擬議清冊.html
│   └── 家長會獎學金印領清冊.html
└── output_115/           # 生成結果（自動建立）
    ├── 115年_獎懲申請表_黃云毅.html
    ├── 115年_獎懲申請表_蕭睿彭.html
    ├── 115年_獎懲申請表_呂善衡.html
    ├── 115年_敘獎擬議清冊.html
    └── 115年_家長會獎學金印領清冊.html
```

## 如果要請 AI 幫你更新名單

直接把獲獎名單截圖或貼文字給 AI，說：

> 「請幫我更新 winners.yaml，今年得獎的是...」

AI 會直接修改 `winners.yaml`，你再執行 `python gen.py` 即可。
