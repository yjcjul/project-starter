# 🏫 共用知識庫（_base）

> 本資料夾存放學校基礎環境的通用資訊。  
> 這些資訊會在建立新專案時自動複製到新專案中。

## 用途

- 記錄學校的**不會隨專案改變**的基礎資訊
- 每次建立新專案時自動帶入
- 專案獨立出去時也會帶走一份副本
- 如果在任何專案中更新了這些資訊，記得同步回 `project-starter/_base/`

## 檔案清單

| 檔案 | 內容 |
|------|------|
| `school-network.md` | 校園網路架構（VLAN、交換機、機房） |
| `school-hardware.md` | 校內硬體設備清單（桌機型號、規格） |
| `school-infra.md` | 學校基礎設施（AD、伺服器、機房） |
| `school-overview.md` | 學校基本概況（教室/辦公室分佈） |
| `school-vlan-table.md` | 完整 VLAN 對照表（含位置、IP、用途） |

## 更新原則

1. 任何專案中發現新的基礎資訊 → 更新到該專案的 `_base/` 中
2. 定期將各專案的 `_base/` 同步回 `project-starter/_base/`（保持最新）
3. `_base/` 中只放**通用、跨專案**的資訊，專案特有的放在各自的資料夾
