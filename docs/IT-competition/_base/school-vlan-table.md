## VLAN 命名規則整理

```
一號網路：VLAN 2~99   → IP 192.168.N.xxx     → Edge SW 192.168.8.N
二號網路：VLAN 102~199 → IP 192.168.N.xxx（推測）→ Edge SW 192.168.108.N
三號網路：VLAN 200~253 → 用途較特殊（AP、監控、LED等）
```
