| **ESXi 主機 1** | UCS C240 M3 等 退役 | **R760** | ESXi 8.0 | 2031 |

| **ESXi 主機 3** | | **R630 #1** | ESXi 7.0 | 2027(裸跑) |

> VMware Essentials 限制最多 3 台 ESXi 主機，剛好用滿。
> DataCore 兩台之間透過 10GbE 網路做 Mirror 同步。
> ESXi 透過 FC (經 Brocade SAN Switch) 掛載 DataCore 提供的 LUN。

### 新架構圖

```
  R720xd#1 (Storage)            R720xd#2 (Storage)
  ──[SAS]→ MD3420#1             ──[SAS]→ MD3420#2
  ──[SAS]→ MD1220新
  ──[FC]→ Brocade #1            ──[FC]→ Brocade #2
             │    │                       │    │
             ▼    ▼                       ▼    ▼
     R760(DataCore主)             R740xd(DataCore Mirror)
     [FC×2] [10GbE]               [10GbE] [FC×2]
       │       └───── Mirror同步 ────┘       │
       │                                     │
       └──[FC]→ Brocade #1         Brocade #2 ←[FC]──┘
                  │    │              │    │
                  ▼    ▼              ▼    ▼
                R760  R660  R630#1  (每台2張FC HBA
               (ESXi) (ESXi)(ESXi)  雙路徑到2台Brocade)
```
