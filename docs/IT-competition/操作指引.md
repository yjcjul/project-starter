# 資訊競賽年度業務操作指引

> 臺中市立黎明國民中學　教務處
> 每年競賽結果出爐後，依本指引操作，約 **15 分鐘** 即可完成所有文件。

---

## 一、業務時程

| 時間點 | 工作項目 |
|--------|----------|
| 競賽結果公布後 | 更新 `winners.yaml`，生成成績公告便簽 |
| 收到教育局公文後 | 補填公文字號，生成敘獎便簽與所有附件 |
| 獎懲會議後 | 列印獎懲申請表，進行簽核流程 |

## 三、每年操作步驟

### Step 1：更新獲獎資料

開啟 `tool/winners.yaml`，修改以下欄位：

```yaml
year: 116                          # 改成當年民國年
competition: 116年度中小學資訊網路應用競賽
doc_ref: "中市教課字第1160012345號"  # 收到教育局公文後填入

students:
  - name: 學生姓名
    class: "班級"       # 例如 "807"（字串加引號）
    seat: "座號"        # 例如 "10"
    event: 競賽項目名稱
    rank: 優等          # 優等 / 甲等 / 乙等
    award: 小功壹次     # 獎懲項目（見下方對照表）
    prize: 200          # 家長會獎學金（元）
    teacher: 指導老師姓名

teachers:               # 只列有資格嘉獎/小功的老師，僅獎狀者不列
  - name: 老師姓名
    title: 教師
    contribution: 指導學生XXX獲XXX項目優等
    award: 嘉獎1次
    prize: 200
```

#### 敘獎額度對照表（縣市級）

| 名次 | 學生獎懲 | 指導老師 | 家長會獎學金 |
|------|----------|----------|-------------|
| 優等 | 小功壹次 | 嘉獎壹次 | 200元 |
| 甲等 | 嘉獎貳次 | 獎狀（不列敘獎） | 150元 |
| 乙等 | 嘉獎壹次 | 獎狀（不列敘獎） | 100元 |

> 依據：學校「學生參加各類競賽敘獎原則」第五點、家長會獎學金實施辦法

---

### Step 2：生成 HTML 文件

在 VS Code 終端機執行：

```powershell
& "C:\projects\procurement-mcp\.venv\Scripts\python.exe" "C:\projects\project-starter\docs\IT-competition\tool\gen.py"
```

生成的檔案會放在 `output_116/` 資料夾。

---

### Step 3：撰寫便簽（公文系統）

複製 `output_116/116年_成績公告簽核.md` 與 `116年_敘獎簽核.md` 的內容貼入公文系統。

**成績公告便簽**（競賽結果出爐後送出）：
- 報告本校參加競賽得獎情形
- 附件：得獎名冊

**敘獎便簽**（收到教育局公文後送出）：
- 附件1：競賽實施計畫
- 附件2：敘獎原則
- 附件3：家長會獎學金辦法
- 附件4：敘獎擬議清冊
- 會辦單位：在公文系統中選擇（總務處、家長會、學務處、人事室）

---

### Step 4：列印紙本文件

用瀏覽器開啟 `output_116/` 中的 HTML 檔案，按 `Ctrl+P` 列印：

**決賽文件**

| 文件 | 列印設定 | 用途 |
|------|----------|------|
| 獎懲申請表（每人一份）| A4 直向、縮放100% | 獎懲會議簽核 |
| 敘獎擬議清冊 | A4 **橫向**、縮放100% | 附件4 |
| 家長會獎學金印領清冊 | A4 直向、縮放100% | 家長會核章 |

**複賽文件**（需填入 `preliminary` 資料後才會生成）

| 文件 | 列印設定 | 用途 |
|------|----------|------|
| 複賽座位表_（學生姓名）.html（每人一份）| A4 直向、縮放100% | 學生出賽前確認位置；第2頁為比賽辦法 |
| 複賽公假單_（競賽項目）.html（每項目一份）| A4 直向、縮放100% | 導師簽名後留存 |

> 列印時勾選「背景圖形」

## 四、複賽文件補充說明

複賽文件在 Step 1 更新 `winners.yaml` 時，需額外填寫 `preliminary` 區段：

```yaml
preliminary:
  date: "116年3月XX日"             # 比賽日期（主辦單位公告後填入）
  location: 電腦教室二
  contact_teacher: （老師姓名）     # 賽前讓學生到教務處找的老師
  fill_date: "116年3月XX日"        # 公假單填表日期（比賽前1~2天）
  report_dates:                     # 賽前兩次報到時間
    - "3月XX日(星期X)中午時間"
    - "3月XX日(星期X)中午時間"
  room_rows: 2                      # 教室座位行數（主辦單位提供平面圖）
  room_cols: 8                      # 教室座位列數

  events:
    - id: typing_en
      name: 電腦打字國中英文甲組
      leave_date: "116年3月XX日"    # 公假單日期
      leave_start_period: "第3節"
      leave_end_period: "第4節"
      prep_time: "9:50至電腦教室"
      competition_time: "10:20~11:00"
      pre_checklist: [安裝自備鍵盤, 測試學校鍵盤]
      rules_scoring: ""             # 電腦打字無計分說明，留空
      rules: [...]                  # 複製當年比賽辦法內容
      students:
        - name: 學生姓名
          class: "807"              # 班級（字串加引號）
          seat_no: "10"
          account: "帳號"           # 主辦單位報名後提供
          password: "密碼"
          room_row: 1               # 教室座位位置（主辦單位座位表提供）
          room_col: 6
```

**生成的複賽文件：**
- `複賽座位表_（姓名）.html`：每人一份，第1頁為座位圖（自己黑格），第2頁為比賽辦法
- `複賽公假單_（競賽項目）.html`：同一項目全部學生共一張

**比賽辦法更新方式：** 每年主辦單位會公告新的比賽辦法，將內容更新到 `winners.yaml` 的 `rules` 欄位即可，無需修改模板。

## 五、注意事項

1. **同一老師指導多位學生**：家長會獎金只領一份（gen.py 已自動去重）
2. **甲等指導老師**：依競賽計畫只有獎狀，不列入敘獎擬議清冊，但仍列入家長會印領清冊
3. **班級格式**：winners.yaml 中用 `"807"` 格式（字串加引號避免前置零消失）
4. **公文字號**：收到教育局公文前，`doc_ref` 先填 `"（待填入）"`，收到後再補填並重新執行 gen.py
5. **便簽中同一老師指導多位學生**：合併為一項敘述，例如「陳怡蓁老師指導7年1班呂善衡同學與7年9班蕭睿彣同學獲…」
