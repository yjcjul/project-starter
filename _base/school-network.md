# 🌐 校園網路架構

> 📅 最後更新：2026-04-16

---

## 架構概述

學校有**兩組獨立的網路**，各有自己的核心交換機：

```
機房
├── 【一號網路】Core SW: 192.168.8.254
│   ├── Edge SW 192.168.8.24 → VLAN 24（教室桌機）→ 192.168.24.xxx
│   ├── Edge SW 192.168.8.25 → VLAN 25（教室桌機）→ 192.168.25.xxx
│   ├── VLAN 3（監視攝影機等特殊設備）→ 192.168.3.xxx
│   └── 其他 VLAN ...
│
├── 【二號網路】Core SW: 192.168.108.254
│   ├── Edge SW 192.168.108.24 → VLAN 248（AP + 公播系統）
│   ├── 其他 Edge SW ...
│   └── AP 的 DHCP 由教育局 Controller 派發
│
├── SAN Switch（✅ 正常）
└── 其他設備
```

## 交換機設備

全部為 **Cisco**，主要型號：

| 型號 | 角色 | 數量 | 備註 |
|------|------|------|------|
| Cisco Catalyst 4500X | 一號網路核心交換機 | 1 台 | 192.168.8.254 |
| Cisco Nexus 3524X | 二號網路核心交換機 | 1 台 | 192.168.108.254 |
| Cisco C3750X | 邊緣交換機 | 多台 | 主力機型 |
| Cisco 2960 | 邊緣交換機 | 少數 | |

> ✅ 交換機全部正常運作。  
> 大部分交換機都有設定 **trunk port**。

## 兩組網路對照

| | 一號網路 | 二號網路 |
|---|---------|---------|
| 核心交換機 IP | 192.168.8.254 | 192.168.108.254 |
| 主要用途 | 桌機、監控、一般設備 | AP、公播系統 |
| Edge SW IP 規則 | 192.168.8.X | 192.168.108.X |
| DHCP | ❌ 目前壞掉（校內自建） | 教育局 Controller 派發 |
| 範例 | SW 192.168.8.24 → VLAN 24 | SW 192.168.108.24 → VLAN 248 |

## VLAN 與 IP 對應規則

**一號網路**的命名規則非常清晰：

```
VLAN N → IP 網段 192.168.N.xxx
```

| VLAN | IP 網段 | 用途 | 備註 |
|------|---------|------|------|
| VLAN 3 | 192.168.3.xxx | 監視攝影機等特殊設備 | |
| VLAN 24 | 192.168.24.xxx | 教室桌機（棟B） | Edge SW: 192.168.8.24 |
| VLAN 25 | 192.168.25.xxx | 教室桌機（棟A） | Edge SW: 192.168.8.25 |
| VLAN 248 | ❓ | AP + 公播系統（二號網路） | DHCP 由教育局派發 |

> ❓ 完整的 VLAN 表待補充（使用者有一份 VLAN 表）

## 可用於快速查詢的指令

> 如需提供交換機設定給 AI 參考，可以在核心交換機上執行：

```
show vlan brief
show ip interface brief
show cdp neighbors
show running-config | include interface|vlan|ip address
show spanning-tree summary
```
